FreeCodeCamp API Basejump: URL Shortener Microservice

What do I do:
    1) If you pass me an URL as a parameter, you will receive a shortened URL as a JSON response
    2) When you visit that shortened URL, I will redirect you to your original link

Example creation usage:

https://mat-shorturl.glitch.me/new/https://www.google.com
https://mat-shorturl.glitch.me/new/http://foo.com:80

Example creation output:

{ "original_url":"https://www.google.com", "short_url":"https://mat-shorturl.glitch.me/x61rp" }

Usage:

https://mat-shorturl.glitch.me/x61rp

Will re-direct to:

https://www.google.com
